기능 - Record Picker

Record Picker가 컬렉션을 위해 할 수 있는 모든 것.

App Store지원 문의

Record Picker v1.2

Record Picker는 카탈로그화, 스마트 선택, 데이터 보강, 통계, 백업, 시스템 연동을 한데 모아 음악 컬렉션에 생기를 되찾아 줍니다.

아래 페이지는 첫 가져오기부터 다음 레코드 선택까지, 아트워크, 필터, 데이터 품질 점검, 내보내기 도구를 포함한 v1.2의 주요 기능을 소개합니다.

기능

완전한 음악 카탈로그

시작 데이터가 불완전해도 사용할 수 있는 컬렉션을 만드세요.

  • CSV 컬렉션 가져오기
  • 바이닐, CD, 앨범 수동 입력
  • 카메라로 바코드 스캔
  • MusicBrainz와 Discogs로 메타데이터 검색
  • 정보를 채우기 위한 편집 가능한 레코드 상세 시트

아트워크와 시각 데이터

직접 가져오거나 선택한 아트워크로 컬렉션 탐색을 즐겁게 유지하세요.

  • Cover Art Archive로 아트워크 검색
  • 사진 또는 파일에서 수동 가져오기
  • 아트워크가 없을 때 웹 검색
  • 원본 아트워크 복원
  • 누락된 아트워크와 데이터 품질 점검

스마트 선택

보관함에서 무엇이 선택될 수 있는지 통제하면서 다음 레코드를 고르세요.

  • 애니메이션 무작위 선택
  • 연도, 장르, 스타일, 즐겨찾기, 사용 가능 여부로 필터링
  • 같은 아티스트 제외
  • 즐겨찾기를 우선하거나 즐겨찾기만 선택
  • 레코드, 아티스트, 조건의 임시 제외

분위기와 재생

Record Picker는 분위기에 맞게 선택을 이끌고 무엇을 재생했는지 기록합니다.

  • 사용 가능한 경우 로컬 Apple 모델로 분위기 기반 선택
  • 모델을 사용할 수 없을 때 로컬 메타데이터 기반 매칭
  • 최근 선택되거나 재생된 레코드 기록
  • 가능한 경우 Apple Music으로 재생
  • 원하는 레코드를 가까이 두는 위시리스트

탐색과 컬렉션 인사이트

앱을 떠나지 않고 컬렉션을 탐색하고 확인하고 이해하세요.

  • 검색, 정렬, 필터가 있는 레코드 보관함
  • iPhone과 iPad에 맞춘 화면
  • 큰 커버와 빠른 목록
  • 컬렉션, 선택, 즐겨찾기, 제외 통계
  • 누락된 트랙, 포맷, 레이블, 날짜, 데이터 감지

백업, 자동화, 개인정보

고급 도구는 사용자의 통제 아래 있으며 컬렉션은 로컬에 저장됩니다.

  • 컬렉션 백업과 복원
  • 읽기 쉬운 CSV와 JSON 내보내기
  • Siri 단축어와 시스템 동작
  • Apple Watch 연동
  • 사용자 계정, 광고, 컬렉션을 저장하는 Record Picker 서버 없음
