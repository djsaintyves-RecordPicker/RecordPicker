Record Picker v1.2

Record Picker samler katalogisering, smarte valg, databerikelse, statistikk, sikkerhetskopier og systemledsagere for å gi musikksamlingen nytt liv.

Siden nedenfor viser hovedfunksjonene i versjon 1.2, fra første import til neste platevalg, inkludert omslag, filtre, datakvalitet og eksportverktøy.

Funksjoner

Komplett musikkatalog

Bygg en brukbar samling selv når startdataene er ufullstendige.

  • CSV-import av samling
  • Manuell registrering av vinyl, CD og album
  • Strekkodeskanning med kameraet
  • Metadataoppslag med MusicBrainz og Discogs
  • Redigerbare platedetaljer for å fullføre informasjon

Omslag og visuelle data

Hold samlingen behagelig å bla i med omslag du henter eller velger selv.

  • Omslagssøk via Cover Art Archive
  • Manuell import fra Bilder eller Filer
  • Weboppslag når omslag mangler
  • Gjenopprett originalt omslag
  • Kvalitetskontroller for manglende omslag og data

Smart valg

Velg neste plate uten å miste kontrollen over hva som kan trekkes fra kassen.

  • Animert tilfeldig valg
  • Filtrering etter år, sjangre, stiler, favoritter og tilgjengelighet
  • Utelukkelse av samme artist
  • Prioriter favoritter eller begrens valg til favoritter
  • Midlertidige utelukkelser for plater, artister eller kriterier

Stemninger og avspilling

Record Picker kan styre valg etter stemning og holde oversikt over det du spiller.

  • Stemningsbasert valg med lokale Apple-modeller når de er tilgjengelige
  • Lokal metadatabasert matching når modeller ikke er tilgjengelige
  • Nylig historikk over valgte eller spilte plater
  • Avspilling via Apple Music når tilgjengelig
  • Ønskeliste for å holde ønskede plater lett tilgjengelig

Bla gjennom og forstå samlingen

Bla gjennom, kontroller og forstå samlingen uten å forlate appen.

  • Platekasse med søk, sortering og filtre
  • Visninger tilpasset iPhone og iPad
  • Store omslag og raske lister
  • Statistikk for samling, valg, favoritter og utelukkelser
  • Oppdaging av manglende spor, formater, etiketter, datoer og data

Sikkerhetskopi, automatisering og personvern

Avanserte verktøy forblir under din kontroll, med samlingen lagret lokalt.

  • Sikkerhetskopi og gjenoppretting av samling
  • Lesbare CSV- og JSON-eksporter
  • Siri-snarveier og systemhandlinger
  • Apple Watch-følgeapp
  • Ingen brukerkonto, ingen reklame og ingen Record Picker-server som lagrer samlingen din
