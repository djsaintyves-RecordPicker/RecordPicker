機能 - Record Picker

Record Pickerがあなたのコレクションにできることすべて。

App Storeサポートに連絡

Record Picker v1.2

Record Pickerはカタログ化、スマートな選択、データ補完、統計、バックアップ、システム連携をまとめ、音楽コレクションに新しい命を吹き込みます。

このページでは、初回インポートから次のレコード選び、アートワーク、フィルター、データ品質、エクスポートまで、v1.2の主要機能を紹介します。

機能

完全な音楽カタログ

最初のデータが不完全でも、使えるコレクションを作れます。

  • CSVコレクションのインポート
  • レコード、CD、アルバムの手動追加
  • カメラでバーコードをスキャン
  • MusicBrainzとDiscogsでメタデータ検索
  • 情報を補完できる編集可能なレコード詳細

アートワークと視覚データ

取得したり自分で選んだアートワークで、眺めやすいコレクションを保てます。

  • Cover Art Archiveでアートワーク検索
  • 写真またはファイルから手動インポート
  • アートワークがない場合のWeb検索
  • 元のアートワークを復元
  • 不足しているアートワークとデータの品質チェック

スマートな選択

箱から選ばれる候補をコントロールしたまま、次のレコードを選べます。

  • アニメーション付きランダム選択
  • 年、ジャンル、スタイル、お気に入り、利用可否でフィルター
  • 同じアーティストの除外
  • お気に入りを優先、またはお気に入りだけに限定
  • レコード、アーティスト、条件の一時的な除外

ムードと再生

Record Pickerは気分に合わせて選択を導き、再生したものを記録できます。

  • 利用可能な場合はAppleのローカルモデルでムード選択
  • モデルが使えない場合はメタデータでローカル照合
  • 選ばれた/再生したレコードの最近の履歴
  • 利用可能な場合はApple Musicで再生
  • 欲しいレコードを手元に置くためのウィッシュリスト

閲覧とコレクション分析

アプリを離れずにコレクションを閲覧、確認、理解できます。

  • 検索、並べ替え、フィルター付きレコード箱
  • iPhoneとiPadに最適化されたビュー
  • 大きなジャケットと高速リスト
  • コレクション、抽選、お気に入り、除外の統計
  • 不足している曲、形式、レーベル、日付、データの検出

バックアップ、自動化、プライバシー

高度なツールも自分で管理でき、コレクションはローカルに保存されます。

  • コレクションのバックアップと復元
  • 読みやすいCSV/JSONエクスポート
  • Siriショートカットとシステムアクション
  • Apple Watchコンパニオン
  • ユーザーアカウントなし、広告なし、コレクションを保存するRecord Pickerサーバーなし
