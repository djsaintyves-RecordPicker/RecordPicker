Mac 应用

Mac 版 Record Picker

安静、原生地整理黑胶，并决定今晚播放哪张唱片。

获取 iPhone 和 iPad 应用查看当前功能

Record Picker 将你的唱片收藏带到 Mac：快速目录、按氛围推荐的引擎，以及自动补充乐评内容，全部呈现在干净的原生 SwiftUI 应用中。

无广告、无追踪、无订阅。你的资料库通过 iCloud 在 Mac、iPhone 和 iPad 之间私密同步。

优雅建档

以可缩放封面马赛克或可排序列表浏览收藏。直接编辑任意字段：标题、艺人、流派、标签、年份、格式、厂牌、国家、备注和自定义封面。

即时找到一切

全文搜索包含实时结果导航、匹配高亮和逐项浏览。八个分区可通过 Command-1 到 Command-8 打开。

按心情选唱片

输入一个时刻，例如 rainy Sunday afternoon、electro night 或 Britpop。Mood Pick 会从你的架上推荐合适的专辑并避免重复。

亮点

  • 自动乐评补充。 从 Apple Music Editor's Notes、Last.fm、Wikipedia 和 Discogs 填入评论备注。选择来源顺序、合并来源，并保护手写评论。
  • 保持资料库整洁。 数据质量面板会标出重复项、缺失封面、年份、格式、流派、评论等，并支持一键和批量修复。
  • 随处聆听。 直接从唱片卡片打开 Apple Music、Spotify、Deezer、Tidal、Qobuz、YouTube Music、Amazon Music 或 SoundCloud 中的专辑。
  • 统计与洞察。 查看主要艺人、厂牌、年份和格式，并可使用 Apple Intelligence 生成叙述式分析。
  • 隐私优先设计。 SwiftData 和 CloudKit 在设备之间同步收藏、自定义封面和历史记录，无需账号，也无需第三方服务器。
  • 导入与备份。 从 CSV 导入，包括 Discogs 兼容文件，并以 JSON 备份和恢复。Record Picker 已本地化为 25+ 种语言。

要求

  • 运行于 Apple silicon 和 Intel 的 macOS。
  • Apple Music 乐评补充需要 Apple Music 订阅和授权。
  • Last.fm 补充使用免费的个人 API 密钥。
